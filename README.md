# ✈️ FlyMate

This project is a full-featured **Airline Booking System** with both customer-facing features for flight booking and an admin panel for managing flights, employees, and reporting. Built with Python Django and a modern HTML/JavaScript interface, it provides a smooth and secure user experience for both passengers and admins.

---

## 🚀 Features

### 🧑‍💼 Admin Panel Features

**✈️ Flight Management**
- Add new flights: flight number, source, destination, timings, seats, price.
- Update or cancel flights.

**👨‍💻 Employee Management**
- Add and manage employee records.
- Update contact info and roles.
- Generate employee performance/activity reports.

**📊 Report Generation**
- Generate reports (sales, booking trends, employee activity).
- Export reports to PDF or Excel.

---

### 👤 Customer Features

**📝 User Registration & Authentication**
- Register with name, email, phone number, and password.
- Secure login system.

**🔍 Flight Search & Booking**
- Search flights by source, destination, and date.
- View flight details, pricing, and seat availability.
- Book tickets with seat selection and get a unique booking ID.

**💳 Payment Integration**
- Razorpay-based secure payment processing.

**📅 Booking Management**
- View upcoming and past bookings.

---

### 🔔 Notifications & Alerts
- Email/SMS notifications for:
  - Booking confirmation
  - Payment receipts
  - Refund status

---

## 💻 Tech Stack

| Layer         | Technology                     |
|--------------|---------------------------------|
| **Frontend** | HTML, CSS, JavaScript           |
| **Backend**  | Python (Django Framework)       |
| **Database** | MongoDB Atlas                   |
| **Payments** | Razorpay                        |
| **Web Server** | Localhost (Dev), Django server |
| **OS**       | Windows                         |
