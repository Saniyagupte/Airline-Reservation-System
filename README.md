# ✈️ FlyMate

**FlyMate** is a full-featured Airline Booking System with both customer-facing features and an admin dashboard for efficient flight, employee, and report management. Built with Python Django and a modern HTML/JavaScript interface, it provides a smooth, secure experience for passengers and admins alike.

---

## 🚀 Features

### 🧑‍💼 Admin Panel Features

**✈️ Flight Management**
- Add, update, or cancel flights with details like flight number, source, destination, timings, seats, and price.

**👨‍💻 Employee Management**
- Add and update employee records (contact info, roles).
- Generate performance or activity reports.

**📊 Report Generation**
- Generate reports: sales, booking trends, employee activity.
- Export reports in PDF or Excel formats.

---

### 👤 Customer Features

**📝 User Registration & Authentication**
- Sign up with name, email, phone, and password.
- Secure login functionality.

**🔍 Flight Search & Booking**
- Search for flights by source, destination, and date.
- View detailed flight info: timing, price, seat availability.
- Book tickets with seat selection and unique reservation ID.

**💳 Payment Integration**
- Razorpay integration for secure online payments.

**📅 Booking Management**
- View upcoming and past bookings.

---

### 🔔 Notifications & Alerts

- Email or SMS notifications for:
  - ✅ Booking confirmations  
  - 💸 Payment receipts  
  - 🔁 Refund status  

---

## 💻 Tech Stack

| Layer         | Technology                     |
|--------------|---------------------------------|
| **Frontend** | HTML, CSS, JavaScript           |
| **Backend**  | Python (Django Framework)       |
| **Database** | MongoDB Atlas                   |
| **Payments** | Razorpay                        |
| **Web Server** | Localhost (Dev), Django server |
| **OS**       | Windows                         |
